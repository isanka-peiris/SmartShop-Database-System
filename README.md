# SmartShop Ltd - Database Design and Implementation

This repository contains the SQL scripts and NoSQL concepts for the CIS5004 Data and Knowledge Management assignment.

## Instructions to Obtain and Run the Project
1. Download this repository to your computer as a ZIP file or clone it.
2. Open **Microsoft SQL Server Management Studio (SSMS)**.
3. Open and execute the `01_Table_Creation.sql` file to create the `SmartShopDB` database and its tables.
4. Open and execute the `02_Data_Insertion.sql` file to populate the database with sample data.
5. Open and execute the `03_Key_Queries.sql` file to generate business insights (Top-selling products, sales by branch, etc.).
